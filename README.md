This algorithm initially opens a window for the user to choose the intended folder.
The folder should have .ims files in which the user wants to perform a Maximum Intensity Projection.

The algorithm will perform the Maximum Intensity Projection in one file at a time.
All files will be saved in the folder "\maximum intensity projections" that will also be created inside the chosen folder.
